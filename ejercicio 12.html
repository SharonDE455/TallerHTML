LOS LENGUAJES DE PROGRAMACION MAS USADOS EN EL MUNDO

  • C++

    Es un lenguaje de programación que proviene de la extensión del lenguaje C para que pudiese
    manipular objetos y es utilizada en programación de videojuegos en Unity

  • Python

    Es ampliamente utilizado por empresas de todo el mundo para construir aplicaciones web,
    analizar datos, automatizar operaciones y crear aplicaciones empresariales fiables y escalables.

  • Javascript

    Es un lenguaje de programación que puede ser utilizado para crear programas que son integrados a una
    página web o dentro de aplicaciones más grandes.

  • Java

    Sirve para crear aplicaciones y procesos en una gran diversidad de dispositivos
    Se base en programación orientada a objetivos
